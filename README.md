# Notas
Programa de multiplicacion con Node.js

```
Opciones:
      --help     Muestra ayuda                              [booleano]
      --version  Muestra número de versión                  [booleano]
  -b, --base     Base de la tabla                           [número] [requerido]
  -l, --list     Lista la tabla en consola                  [booleano] [defecto: false]
  -d, --desde    Desde que numero empieza en multiplicador  [número] [defecto: 1]
  -h, --hasta    Hasta que numero termina el multiplicador  [número] [defecto: 10]
  ```
